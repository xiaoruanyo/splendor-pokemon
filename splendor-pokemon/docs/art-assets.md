# 游戏美术素材清单

> 用 Gemini 或其他 AI 图片生成工具，按提示词生成素材。
> 下载后放入 `splendor-pokemon/public/assets/ui/` 目录。

---

## 目录准备

先在项目里创建素材目录：

```
splendor-pokemon/public/assets/ui/
```
